### gnuplot is amazing!


to start gnuplot prompt:

``` bash
gnuplot
```
to test gnuplot installation
``` bash
set terminal dumb
plot sin(x)
```

to set terminals:

``` bash
set term
set terminal
```

to save output as a file use:

``` bash
set output "output-file-name.extension"
```

``` bash
set terminal svg
set output "output.svg"
```
``` bash
set term eps
set output "output.eps"
```
